# Carpeta `javascript`

Descripción de los archivos contenidos en esta carpeta:

- **descarga-ppt-av.js**: Código base para descargar todos los archivos PPT en el aula virtual (aula virtual / plataforma AV).
- **llenar-encuesta-docente.js**: Script que rellena automáticamente la encuesta docente del intranet.

Advertencia:

Este código no es limpio ni está preparado para producción. Está hecho únicamente para copiar y pegar en la consola del navegador. Úsalo bajo tu propia responsabilidad y verifica manualmente cualquier acción antes de ejecutarla.

Uso recomendado:

1. Abre la página correspondiente en el navegador (aula virtual o intranet).
2. Abre la consola del desarrollador (F12 / Ctrl+Shift+I).
3. Pega el contenido del archivo JavaScript correspondiente y ejecútalo.
